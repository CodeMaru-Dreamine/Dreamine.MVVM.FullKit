## 프로젝트 소개

.NET, Blazor, WPF와 서비스 운영 경험을 프로젝트·이력·기술 스택 단위로 보여주는 개발자 포트폴리오입니다.

프로젝트 데이터, 갤러리·동영상, 이력서, 검색·필터와 OG 공유 화면을 제공합니다.

## 주요 기능

- 프로젝트 카드 검색·카테고리 필터
- 다중 이미지와 동영상 갤러리
- GitHub·라이브·문서 링크
- 기술 스택·경력·학력 타임라인
- 보호가 필요한 이미지 단계별 블러

## 이용 순서

1. 홈에서 검색어와 카테고리로 프로젝트를 찾습니다.
2. 상세 페이지에서 이미지·동영상과 사용 기술을 확인합니다.
3. GitHub·Live·문서 링크로 결과물을 엽니다.
4. 이력서 섹션에서 경력과 기술 스택을 확인합니다.
