## Overview

A developer portfolio presenting .NET, Blazor, WPF, and service-operation experience through projects, history, and skills.

Provides project data, galleries and video, resume content, search and filters, and Open Graph sharing.

## Key features

- Project search and category filters
- Multi-image and video galleries
- GitHub, live, and documentation links
- Skills, career, and education timelines
- Tiered blur protection for sensitive images

## How to use

1. Find projects using search and category filters.
2. Inspect media and technologies on a project detail page.
3. Open GitHub, live, or documentation links.
4. Review career and skills in the resume section.
