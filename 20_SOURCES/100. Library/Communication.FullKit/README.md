# Dreamine.Communication.FullKit

All-in-one meta package for Dreamine Communication core, socket, serial, and broker adapter packages.

## Project information

- Version: 1.0.1
- Target framework: net8.0

## Generate documentation

Run the following command from this project directory:

`powershell
doxygen Doxyfile.en
`

Korean documentation can be generated with Doxyfile.kr.
